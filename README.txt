# Jeu du Snake en Java

Il s'agit d'un jeu simple du Snake implémenté en Java. Le jeu est construit en utilisant Java Swing pour l'interface utilisateur graphique.

## Fonctionnalités

- Jeu du Snake classique où le joueur contrôle un serpent pour manger de la nourriture et grandir en longueur.
- Le jeu se termine si le serpent entre en collision avec les limites du jeu ou avec son propre corps.
- Le joueur peut contrôler le serpent à l'aide des touches fléchées ou des touches WASD.
- Le score du joueur augmente pour chaque élément de nourriture mangé.
- Le jeu affiche le score actuel du joueur et le score le plus élevé atteint.
